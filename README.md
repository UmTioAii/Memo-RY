# MemoRY

Aplicação de memos e tarefas com interface minimalista, foco em produtividade e armazenamento local no navegador.

## Funcionalidades

- Criação, edição e remoção de memos
- Marcação de memos como concluídos
- Filtros por status (todos, ativos e concluídos)
- Visualização em lista e em board com colunas
- Reordenação de memos por arrastar e soltar
- Seleção de cores de marcação
- Persistência local dos dados
- Suporte a tema claro/escuro
- Suporte a múltiplos idiomas

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React Query
- Vitest
